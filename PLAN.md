This is a tvOS application that uses Invidious API to read videos.
The app should have 3 tabs :

- Home
- Settings
- Search

# Tech stack

Always use Expo SDK if possible including routes.
Use ky as http client.
Choose the best UI library that fit the tvOS design.

# Home

Present channels latests videos by channel name.
When a video is pressed, it should open the video player.

# Settings

A simple form that register the Invidious API config required to run this project.
It should be stored in the storage.

# Search

A simple search field that use the invidious search api.
Can be implemented in last.
